# solving for #campaigns by fixing the # of (other) campaigns

def n_social(n_tv, budget):
    return (budget-250*n_tv)/25

def n_tv(n_social, budget):
    return (budget-25*n_social)/250

visual inspection¶

  • red line = max budget

math solver¶

  • solve for finding revenue contour = tangential to constraint line, using lagrange multiplier.
  • this means finding point where revenue contour gradient is proportional (not equal to) to constraint gradient.
  • mathematically, "gradient of revenue" = lambda * "gradient of constraint".

gradients are derivatives¶

  • we need the derivatives of the revenue & constraint functions. they each have two variables, so we need two partial derivatives.
In [13]:
from sympy import *

s, t, l = symbols('s t l')

solve([
    Eq((21/4)*((t**(1/4))/s**(1/4)) -  25*l, 0),
    Eq(( 7/4)* (s**(3/4) /t**(3/4)) - 250*l, 0),
    Eq(25*s+250*t - 2500, 0)], [s,t,l], simplify=False)
